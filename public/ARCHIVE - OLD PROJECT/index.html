How To


Die Wikimedia Foundation will ihr Onlinelexikon Wikipedia möglichst neutral halten. Das dies nicht immer einfach ist zeigt sich am Beispiel sogenannter "Edit-Wars" - Kriege, die zwischen Autoren, um den Inhalt eines Wikipediaeintrags ausgetragen werden. Häufig finden diese Kriege parallel zu aktuellen internationalen Konflikten statt.


Unser Ansatz: Extraction von Wikipedia Revisionlogs, Aufbereitung und Anreicherung der Daten in Java mittels Wordvektoren, IP-Geolokation uvm. Visualisierung der Daten als interaktive Website mittels Javascript und CSS3.


Unsere Vision: Eine live Visualisierung aller Änderungen eines Artikels als Heatmap nach Region sowie die automatische Extraktion kontroverser Teile eines Artikels.
